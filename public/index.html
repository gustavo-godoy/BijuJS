Getting Started

Using BijuJS is very easy!

Open your browser's developer tools console and type in:

BijuJS.hello("world", console.log)

This calls run/hello with "world" argument and console.log as callback.

The function returns a Promise so you can check your request's status. Then right after you should get a response object that looks like this:

{ "status": "0, "stdout": "Hello, world! It works!\n", "stderr": "" }

So, say you want to set up a script named example, then:

  1. Add example inside the run/ folder
  2. Create your front-end page on the public/ folder
  3. Call it in your page as BijuJS.example([args,], callback)

That's it!

What's next?

BijuJS is provided bare-bones, but it's easy to expand upon it.

Consider using MVP.CSS or installing a template rendering engine to quickly develop nice and functional interfaces.
